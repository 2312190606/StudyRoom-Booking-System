## 我完成的工作

### 1. 设计规范
- **配色方案制定**
  - **主色调**：#4F46E5（蓝紫色），作为品牌主题色，广泛用于主要操作按钮、顶部导航高亮等核心交互元素。
  - **背景辅助**：前台大面积使用 #F8FAFC（浅灰），后台大屏监控等场景使用深色系（#0F172A/深石板灰、#1E293B/内容卡片底色）进行视觉分区。
  - **状态语义色**：#10B981（安全/已签到）、#EF4444（危险/违规/过期）、#F59E0B（提示/积分警告），确保用户对当前座位状态一目了然。
- **字体选择**
  - **中文字体**：默认体系 PingFang SC / Microsoft YaHei，保障所有系统下的阅读自然性；各级标题统一进行加粗和字号拉开层级。
  - **数字与英文**：优先使用现代化无衬线字体 Inter，使得系统统计数据和时间显示更具现代科技感且易于辨识。
- **组件库整理**
  - 基于前台移动端体验，对 **Vant UI** 进行了深度定制，统一并提炼了各种边距（Margin/Padding）与圆角（Border Radius）尺寸。
  - 沉淀了“座位状态图例卡”、“顶部状态横幅通知”、“底部 Tabbar”等全局高频复用的基础组件，并实现了组件在 Figma 中的统一规范管理。
