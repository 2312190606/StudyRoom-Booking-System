## 遇到的问题和解决

### 问题：前端数据获取方式错误
**现象**：后端返回数据正确，但前端始终提示"无法回答"

**分析**：响应拦截器返回 `res.data`，而 Axios 响应体已被解包，导致 `res.data` 为 undefined

**解决**：修改 AIChat.vue 中获取数据的方式
```javascript
// 错误：
const res = await askAI(content)
content: res.data || '抱歉，我现在无法回答这个问题'

// 正确：
const answer = await askAI(content)
content: answer || '抱歉，我现在无法回答这个问题'
```
