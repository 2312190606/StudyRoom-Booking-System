# 自习室座位预约系统 - 前端开发文档

## 一、模块功能

### 1. 用户端模块

#### 1.1 用户认证模块

- **注册**：手机号/邮箱注册
- **登录**：账号密码登录，记住登录状态
- **JWT 令牌管理**：自动刷新 Token

#### 1.2 首页模块

- **公告栏**：轮播图展示系统公告
- **自习室列表**：显示自习室名称、位置、剩余座位数
- **搜索筛选**：按位置、楼层筛选自习室

#### 1.3 座位可视化模块

- **座位图展示**：网格布局渲染座位
- **状态区分**：
    - 绿色：可用座位
    - 红色：已占用
    - 灰色：维修中
    - 黄色：已预约（本人）
- **座位编号**：每个座位显示唯一编号

#### 1.4 预约功能模块

- **快速预约**：选择座位，默认预约 1 小时
- **时间选择**：自定义预约时长（30 分钟起）
- **预约确认**：显示预约规则，二次确认

#### 1.5 我的预约模块

- **预约列表**：展示当前/历史预约记录
- **预约操作**：
    - 取消预约（预约时间到之前随时可取消）
    - 预约延后（延长 30 分钟，限 1 次）
- **状态展示**：待使用、使用中、已完成、已取消、已违约

#### 1.6 签到模块

- **定位验证**：调用地图API确认位置签到
- **签到反馈**：签到成功/失败提示

#### 1.7 个人中心模块

- **个人信息**：头像、昵称、手机号修改
- **密码管理**：修改登录密码
- **学习时长统计**：
    - 日/周/月学习时长
    - 可视化图表展示
- **违约记录**：查看违约次数

#### 1.8 常用座位模块

- **座位收藏**：选择常用自习室和座位
- **一键预约**：快捷按钮，快速预约收藏座位
- **收藏管理**：添加/删除常用座位

---

### 2. 管理员端模块

#### 2.1 仪表盘模块

- **数据统计**：
    - 今日预约量（数字 + 趋势图）
    - 座位利用率（百分比 + 时段分布）
    - 用户总数（新增用户曲线）

#### 2.2 自习室管理模块

- **列表展示**：自习室信息表格
- **增删改查**：
    - 新增自习室（名称、位置、开放时间）
    - 编辑信息
    - 删除自习室
    - 批量操作

#### 2.3 座位管理模块

- **可视化布局**：拖拽式座位配置
- **座位属性**：设置座位编号、是否靠窗、是否有电源
- **批量操作**：批量添加/删除座位
- **状态管理**：设置维修状态

#### 2.4 预约记录管理模块

- **记录查询**：按用户、时间、状态筛选
- **详情查看**：预约详细信息
- **强制取消**：管理员权限取消预约
- **数据导出**：导出 Excel 报表

#### 2.5 用户管理模块

- **用户列表**：展示所有用户信息
- **状态管理**：禁用/启用用户账号
- **用户详情**：查看用户预约记录、违约记录

#### 2.6 公告管理模块

- **公告列表**：展示所有公告
- **发布编辑**：编辑公告
- **上下架**：控制公告显示状态
- **删除操作**：删除过期公告

#### 2.7 轮播图管理模块

- **图片上传**：上传图片至服务器
- **排序管理**：调整轮播图顺序
- **启用/禁用**：控制轮播图显示

#### 2.8 系统配置模块

- **预约规则**：
    - 可预约时段设置（如 7:00-23:00）
    - 最短/最长预约时长
    - 最晚取消时间
- **违约设置**：违约次数限制、惩罚规则
- **开放时间**：设置自习室开放时间段

## 二、技术选型
- **核心框架**：Vue 3
- **UI 组件库**：Vant 4.x（移动端）、Element Plus（桌面端）
- **状态管理**：Pinia 2.x
- **路由管理**：Vue Router 4.x
- **HTTP 客户端**：Axios 1.x
- **工具库**：Lodash、Day.js、ECharts、@vueuse/core

## 四、运行方式

### 1. 环境要求

- **Node.js >= 18.x** 
- **npm >= 9.x 或 pnpm >= 8.x**

### 2. 安装依赖
进入项目目录
```bash
cd study-room-frontend
```
安装依赖（推荐使用 pnpm）
```bash
pnpm install
```
或使用 npm
```bash
npm install
```

### 3. 启动开发服务器
```bash 
npm run dev
```
浏览器访问 http://localhost:5173

### 4. 构建生产版本
```bash
npm run build
```

### 5. 环境配置
修改 `.env.development` 中的 `VITE_BASE_API` 为后端接口地址：
```bash
VITE_BASE_API=http://localhost:8080/api
```
