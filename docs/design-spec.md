# 设计说明

## 配色方案
- **主色 (Primary)**：`#4f46e5` (Indigo-600) - 用于品牌标识、主要按钮、激活状态和核心图标。
- **辅助色 (Secondary)**：`#10b981` (Emerald-500) - 用于表示成功状态、高信誉积分、健康入座率等积极指标。
- **强调色 (Accent)**：`#f43f5e` (Rose-500) - 用于“热门”标签、违约记录、警示信息和重要删除/取消操作。
- **中性色 (Neutral)**：`#f8fafc` (Slate-50) 作为全局背景，`#1e293b` (Slate-800) 用于正文文字，`#94a3b8` (Slate-400) 用于次要说明。

## 字体
- **标题 (Headings)**：系统默认无衬线字体 (System Sans-serif)，加粗 (font-bold)，字间距略微收紧 (tracking-tight)，增强现代化专业感。
- **正文 (Body)**：系统默认无衬线字体 (System Sans-serif)，常规字重 (font-normal)，行高适中，确保在移动端小屏幕上的易读性。

## 核心页面
1. **首页 (Home)**：
   - **自习室发现**：通过卡片流展示自习室，包含实景图、评分及实时余量。
   - **动态统计**：顶部展示全校实时在线人数和入座率，营造学习氛围。
   - **快速搜索**：支持按校区、自习室名称或特定标签进行检索。
2. **选座页面 (Room Details)**：
   - **交互式座次表**：网格化布局展示座位，通过图标和颜色区分（可选、已选、占用）。
   - **靠窗标识**：特殊位置（如靠窗）设有视觉标记。
   - **底部动作条**：选中座位后弹出悬浮确认栏，展示预约详情并执行锁定操作。
3. **预约管理 (Bookings)**：
   - **状态驱动**：根据预约状态（待签到、学习中）展示差异化操作（签到、续约、提前结束）。
   - **历史追溯**：列表展示过往预约记录及其完成状态（已完成、已取消）。
4. **个人中心 (Profile)**：
   - **数据可视化**：利用柱状图展示本周每日的学习时长趋势。
   - **信誉体系**：直观展示信誉积分和违约详情，引导规范使用。

## 交互说明
- **导航切换**：底部 Tab Bar 固定在屏幕下方，点击实现首页、预约和个人中心的快速跳转。
- **页面路由**：使用 `motion` 实现左右平滑切入切出效果，提升应用流畅度。
- **预约锁定**：在选座页点击座位即时高亮，并触发底部操作栏升起；点击“确认预约”后通过 Toast 提供即时反馈。
- **签到反馈**：点击签到按钮后显示 Loading 状态，模拟蓝牙验证过程，成功后更新 UI 状态并弹出成功提醒。
- **图表悬停**：个人中心学习曲线图支持点击/悬停查看具体数值详情。
